# SeaStar

This is the repository for the SeaStar framework. You will find the code under [framework](https://github.com/criscod/SeaStar/tree/master/framework) and the experimental [data](https://github.com/criscod/SeaStar/tree/master/data) under /data.

The complete set of clean links extracted from the [Linked Data Crawl 2014](http://data.dws.informatik.uni-mannheim.de/lodcloud/2014/ISWC-RDB/#toc0) is [here](https://drive.google.com/file/d/0B3W6K8QxmfLnc3FwV1lZd1hVemM/view?usp=sharing).
